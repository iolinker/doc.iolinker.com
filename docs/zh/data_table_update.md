## 数据表更新

更新数据表中的一行或多行数据。每行必须包含 `id` 字段来标识要更新的行，支持单次操作批量更新多行数据。

## 入参

- **Table UID**（必填）：要更新数据的数据表唯一标识。

- **Rows**（必填）：要更新的行对象数组，每个对象**必须**包含 `id` 字段以标识行，以及要更新的字段。

### Rows 示例

```json
[
  {"id": 1, "name": "Alice Updated", "age": 26},
  {"id": 2, "name": "Bob Updated", "age": 31}
]
```

> **注意：**
> - 每个行对象中 `id` 字段为必填，缺少 `id` 字段将导致报错。
> - 仅更新行对象中包含的字段（除 `id` 外），未包含的字段保持不变。
> - 如果表中的列设置了**唯一约束（unique）**，更新为该列已有数据的值将会报错。

## 出参

返回包含更新行 ID 和数量的对象：

```json
{
  "updatedIds": [1, 2],
  "count": 2
}
```

- **updatedIds**：成功更新的行 ID 数组。
- **count**：成功更新的行数。
